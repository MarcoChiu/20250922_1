題目四：創造、插入 DOM

請製作一個按鈕，當點擊時會在 HTML 文件中新增以下內容：

// Todo 待編輯
<button>新增你好世界</button>
// Document Added 新增
<p>你好世界</p>

回答:

        const hellowBtn = document.querySelector('#hellowbtn');
        hellowBtn.addEventListener('click',function(e){
            const createitem = document.createElement('p');
            createitem.textContent="你好世界";
            document.querySelector('.hellowdiv').appendChild(createitem);
        });
